# ✈️ Aircraft Demand Forecasting

항공기 수요를 예측하기 위한 머신러닝 기반 프로젝트입니다.
과거 데이터를 기반으로 미래의 항공 수요를 예측하여, 항공 운영 및 전략 수립에 활용할 수 있습니다.

---

## 프로젝트 개요

본 프로젝트는 시계열 데이터를 활용하여 항공기 수요를 예측하는 모델을 구현하는 것을 목표로 합니다.
데이터의 계절성, 추세(trend), 패턴을 학습하여 향후 수요를 예측합니다.

---

## 주요 기능

* 항공 수요 데이터 전처리
* 시계열 데이터 분석
* 머신러닝 기반 수요 예측
* 예측 결과 시각화

---

## 🧠 사용 모델

* LSTM (Long Short-Term Memory)
  또는
* Random Forest / XGBoost
  또는
* 선형 회귀 (Linear Regression)

---

## 데이터

* 항공기 수요 관련 시계열 데이터 사용
* 날짜, 수요량 등의 정보 포함

※ 데이터는 공개 데이터 또는 자체 수집 데이터를 사용
